# rssrn's instance of the [birdbird](https://github.com/rssrn/birdbird) project

This is a configured site instance of the html viewer available in the birdbird project.
It's customised to showcase bird observations from one person/location.

So, if you are reading this, you probably want to look at [https://github.com/rssrn/birdbird](https://github.com/rssrn/birdbird) -
the README file has details on how to set it up.

This instance of birdbird is customised in [config.js](./dist/config.js).
